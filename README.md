# 📰 Fake News Detection ML

A Machine Learning project that detects whether a news article is **Real** or **Fake** using Natural Language Processing (NLP) and classification algorithms.

## 🚀 Features



- Classifies news as Real or Fake
- Text preprocessing and cleaning
- Feature extraction using TF-IDF
- Machine Learning model training
- Accuracy evaluation
- Easy-to-use prediction system

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- TF-IDF Vectorization
- Jupyter Notebook
